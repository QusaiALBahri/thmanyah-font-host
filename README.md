# Thmanyah Sans font host

This public repository serves the official Thmanyah Sans webfont package through GitHub Pages.

The font was sourced from the locally downloaded official Thmanyah Font Family package, which includes its license PDF. The official font site states that the font is free for personal and commercial use, including websites and applications: https://font.thmanyah.com/

GitHub Pages is configured to publish the `main` branch from the repository root.
